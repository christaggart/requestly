Welcome!

This site is for users and publishers of open access data in the National Capital region (opendataottawa.ca). It is a very early alpha release, so if you experience issues report them here. Today you can publish a request for data to the community, where members can comment and rate the request. In future iterations of this site, publishers and others will be able to post details of known and existing data sources so that community members can rate them for prioritization. Users will then be able to find data sources that have been published.

Request

About Us

We are a community of people from the National Capital region who have an interest in freeing data to support innovation in the public, social and private sectors. This site has been built by and for the community. It is a space for community members to find, unlock, publish and use open data for civic, social and economic purposes. More...
